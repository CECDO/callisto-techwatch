Les outils pour faire une bonne veille technologique

Les réseaux sociaux

Pour mener une veille sur les réseaux sociaux qui soit efficace et obtenir les informations désirées, il faut avant tout définir exactement ce que l’on recherche. Quelles sont les informations désirées ? Pour quels besoins ?

GitHub

GitHub est un service d'hébergement de dépôts Git, mais il ajoute de nombreuses fonctionnalités qui lui sont propres. Alors que Git est un outil en ligne de commande, GitHub fournit une interface graphique basée sur le Web. Il fournit également un contrôle d'accès et plusieurs fonctions de collaboration, comme un wiki et des outils de gestion des tâches de base pour chaque projet.

Visual Studio Code

Visual Studio Code a les fonctionnalités qui incluent la prise en charge du débogage, la mise en évidence de la syntaxe, la complétion intelligente du code, les snippets, la refactorisation du code et Git intégré.

DailyDev.dev

daily.dev est une communauté de développeurs qui se réunissent autour de la découverte et de l'exploration de l'actualité du développement. Rejoignez le mouvement pour créer ensemble de meilleurs logiciels.

TweetDeck

Tweetez comme un pro. L'outil Twitter le plus puissant pour le suivi, l'organisation et l'engagement en temps réel. Atteignez votre public et découvrez le meilleur de Twitter.

Presse

Futura est un portail web de vulgarisation scientifique structuré en cinq rubriques : Sciences, Santé, Tech, Maison et Planète.
